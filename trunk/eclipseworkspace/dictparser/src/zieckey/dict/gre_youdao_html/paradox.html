paradox ['pærədɔks]

  • n.
    • (与通常的见解相对立的)反论，悖论
    • 似矛盾(而可能)正确的论点
    • 自相矛盾的话(或事)
    • 充满矛盾的人(或事)
    • 【物理学】佯谬
以上来源于：《21世纪大英汉词典》

paradox [ 'pærədɔks ]

  • n. (logic) a statement that contradicts itself

    "`I always lie' is a paradox because if it is true it must be false"
