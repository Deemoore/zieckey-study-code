format ['fɔ:mæt]

  • n.
    • (出版物的)版式
    • (出版物的)开本[参较 duodecimo, folio, octavo, quarto]
    • 某事的构成、设计、风格、类型等；(安排或表现的)方式，样式
    • 【计算机】格式
    • 【电视】电视节目的总安排(或计划)
    • (录像带的)制式
  • vt.
    • 规定…的格式(或形状、大小、比例等)；安排…的格局(或规格)；设计…的版面；以一定规格制作(印刷品、书本、数据等)；(电视节目等的)设计
    • 【计算机】
      • 确定(输入或输出)的格式，为…编排格式，使格式化
      • 编制书写与阅读的磁盘
  • vi.
    • 设计一个版式
    • 变形：
    • formatted , formatting
以上来源于：《21世纪大英汉词典》

format [ 'fɔ:mæt ]

  • n.
    • the organization of information according to preset specifications (usually for computer processing)

      同义词：formatting | data format | data formatting

    • the general appearance of a publication
  • v.
    • set (printed matter) into a specific format

      同义词：arrange

    • determine the arrangement of (data) for storage and display (in computer science)
    • divide (a disk) into marked sectors so that it may store data

      "Please format this disk before entering data!"

      同义词：initialize | initialise
