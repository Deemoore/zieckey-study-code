file1 ['fail]

  • vt.
      • 汇订(书信、文件等)
      • 把(文件等)归档(或存卷)：

        • Please file (away) those letters in a drawer.

          请把那些信件归档放入抽屉内。

    • 用电话(或电报)向报社发送(新闻稿子)；为(电台)录制(新闻节目)：

      • He files his story from New York.

        他从纽约发出这条消息。

    • 提出(申请、议案等)：

      • to file a petition

        提出请愿

    • 呈请把(法律文件)备案
    • 提出(离婚诉讼或其他讼案)：

      • to file an application for divorce

        提交离婚申请书

  • vi.
    • 排成纵队前进：

      • to file out of a building

        依次从大楼里走出

    • [美国英语](在预选中)备案作候选人(常与for连用)：

      • to file for county attorney

        登记竞选郡司法局长

    • (为离婚诉讼案)提出申请，申请(常与for连用)：

      • to file for a job

        申请就业

    • 把文件(或资料)归档
  • n.
    • 纸夹，文件夹；档案，案卷，卷宗，公文柜；存放音像资料的橱(或架)
    • 文件(或卡片等)的汇订；归档，存卷；音像资料；【计算机】文件，(计算机程序内储存信息的)电子文件，文档，文卷
    • 纵列[参较 rank]
    • 成纵列的人(或动物、物体)
    • (国际象棋棋盘上的)格子纵列
    • (提升名单中的)名次
    • (记者)发送的一批电讯
    • [古语]名册
  • 短语:
    • a file of men 【军事】派出执行任务的二人小组(一前一后)
    • file and forget [美国口语](把…)搁置一边忘掉

    • in file
      • 鱼贯地，依次
      • 成纵队
    • in single (或 Indian) file 【军事】成一路纵队，成单行
    • on file
      • 存档，归档
      • 被记录下来备查
    • run her file 【狩猎】(兔子)绕来绕去地跑使猎犬迷惑
    • vertical file
      • 直立式档案柜
      • (图书馆等中)供速查的资料(指剪报、图书、小册子等)
    • 变形：
    • filed , fileing

  • n.
    • a set of related records (either written or electronic) kept together

      同义词：data file

    • a line of persons or things ranged one behind the other

      同义词：single file | Indian file

    • office furniture consisting of a container for keeping papers in order

      同义词：file cabinet | filing cabinet

    • a steel hand tool with small sharp teeth on some or all of its surfaces; used for smoothing wood or metal
  • v.
    • record in a public office or in a court of law

      "file for divorce"; " file a complaint"

      同义词：register

    • smooth with a file

      "file one's fingernails"

    • proceed in line

      "The students filed into the classroom"

      同义词：charge | lodge
