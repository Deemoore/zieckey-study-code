latency [ 'leitənsi ]

  • n.
    • (computer science) the time it takes for a specific block of data on a data track to rotate around to the read/write head

      同义词：rotational latency

    • the time that elapses between a stimulus and the response to it

      同义词：reaction time | response time | latent period

    • the state of being not yet evident or active

百科

latency

（潜伏期）从字面上了解其含义是比较困难的，实际上，它表示完全执行一个指令所需的时钟周期，潜伏期越少越好。严格来说，潜伏期包括一个指令从接收到发送的全过程。现今的大多数x86指令都需要约5个时钟周期，但这些周期之中有部分是与其它指令交迭在一起的（并行处理），因此CPU制造商宣传的潜伏期要比实际的时间长。
