cipher ['saifə]

  • n.
    • 密码，暗码
    • 密码电报；密码信；密码文件
    • 密码索引
    • (姓名首字母等的)拼合文字，交织字母；花押字
    • 阿拉伯数字；数字
    • 零(即0)
    • 不重要的人；无价值的东西
  • vi.
    • [罕用语]用数字计算，做算术，运算
    • 使用密码
  • vt.
    • [罕用语]计算，算出
    • 用密码书写；把…译成密码；用暗号表示：

      • He ciphered the telegram in a few minutes.

        他只用了几分钟就把电报译成密码。

    • [美国口语]琢磨，想出：

      • I couldn't cipher out his motives.

        我猜想不出他的动机。

  • 短语:
    • a mere cipher 无名小卒，无足轻重的人；有名无实的人
    • cipher in algorism 傀儡；无足轻重的人；零
    • stand for cipher [口语]微不足道，等于零
以上来源于：《21世纪大英汉词典》

cipher [ 'saifə ]

  • n.
    • a message written in a secret code

      同义词：cypher

    • a mathematical element that when added to another number yields the same number

      同义词：zero | 0 | nought | cypher

    • a quantity of no importance

      同义词：nothing | nil | nix | nada | null | aught | cypher | goose egg | naught | zero | zilch | zip | zippo

    • a person of no influence

      同义词：cypher | nobody | nonentity

    • a secret method of writing

      同义词：cypher | cryptograph | secret code

  • v.
    • convert ordinary language into code

      同义词：encode | code | encipher | cypher | encrypt | inscribe | write in code

    • make a mathematical calculation or computation

      同义词：calculate | cypher | compute | work out | reckon | figure
